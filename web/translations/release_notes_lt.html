Naujausia versija yra 0.7.1.

0.7.1:

This version include some bug fixes, some of them important. It's highly recommended to update.

Fixes:

  • SMPlayer took more than 10 seconds to show when running for the very first time.
  • The links to download subtitles from Opensubtitles were wrong.
  • SMPlayer crashed in the favorite editor when trying to select a file if the KDE open dialog was used.
  • Some translations were missing in the Windows package.

New:

  • By default the screenshots are saved in the user's pictures folder instead of the SMPlayer's config folder.
  • Now it's possible to change the opensubtitles server.
  • Youtube: seeking is slow with flv videos, so now flv videos have the lowest priority.
  • Youtube: now it's possible to search and download videos from youtube. This is provided by an external application (in linux you have to install an independent package: smtube).

0.7.0:

  • Naujas mėgstamų meniu, kuriame galite pridėti savo mėgstamus video, muziką, srautus, youtube vaizdelius... Taipogi įmanoma pridėti papildomus meniu.
  • Youtube palaikymas. Dabar galite atverti tokius url kaip http://www.youtube.com/watch?v=..... naudodami Atverti -> URL dialogą arba nuvilkdami nuorodą iš naršyklės ant smplayer lango.
  • Mplayer2 palaikymas (http://www.mplayer2.org). Tai mplayer atšaka su įdomiomis galimybėmis, pvz. tikslus persukimas, mkv failuose išrikiuojami skyriai ir geresnis sustabdymas (pvz. sustabdžius galima prasukti).
  • Naujas vertimas: kroatų.

0.6.10:

  • Naujas vdpau konfigūravimo dialogas, kuris leidžia pasirinkti kuriuos vdpau kodekus naudoti.
  • Pritaikymas eCS, OS/2 (autorius Silvan Scherrer).
  • Naujas meniu paslėptų subtitrų kanalo pasirinkimui (reikia mplayer >= r32607).
  • Galimybė pasirinkti persukimo būdą (absoliutų arba santykinį).
  • Galimybė rikiuoti grojaraščio įrašus.

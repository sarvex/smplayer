Naujausia versija yra 0.8.2.

smplayer 0.8.2:

  • A skinnable interface has been added. Several skins are included.
  • Support for 6.1 and 7.1 audio output.
  • A "Privacy" section in the preferences dialog has been added. You can disable saving the recent files or URLs.
  • The main window can be moved by dragging the video area.
  • Fix for a freeze that may happen on Windows XP.
  • (Windows) The font cache is created before showing the GUI.

smplayer 0.8.1:

  • Pataisytas Youtube atkūrimas.

smplayer 0.8.0:

  • Pridėtas įrankinės redaktorius. Dabar galite pasirinkti kurie mygtukai bus rodomi įrankių juostose.
  • Nauji video filtrai: gradfun, blur ir sharpen.
  • Dabar galima pakeisti naudotojo sąsają(numatyta, mini, mpc) iškart, nebūtina paleisti iš naujo.
  • sub failai iš opensubtitles vėl turėtų veikti.
  • (Youtube) Trumpų url atpažinimas (tokių kaip ši: http://y2u.be/F5OcZBVPwOA)
  • Geresnis skyrių palaikymas.
  • Ištaisyta klaida: nutolę m3u failai veikia iš mėgstamų meniu ar komandinės eilutės.
  • Vidiniai pakeitimai vienos kopijos pasirinktyje (pereita prie QtSingleApplication).

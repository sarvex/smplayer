最新のバージョンは 0.8.2 です。

smplayer 0.8.2:

  • A skinnable interface has been added. Several skins are included.
  • Support for 6.1 and 7.1 audio output.
  • A "Privacy" section in the preferences dialog has been added. You can disable saving the recent files or URLs.
  • The main window can be moved by dragging the video area.
  • Fix for a freeze that may happen on Windows XP.
  • (Windows) The font cache is created before showing the GUI.

SMPlayer 0.8.1:

  • YouTube の再生に関する修正。

SMPlayer 0.8.0:

  • ツールバーエディターが追加されました。これにより、ツールバーに表示したいボタンやコントロールが選択できるようになりました。(動画を見る)
  • ビデオフィルター [gradfun]、[ブラー]、[シャープ] が新たに追加されました。
  • 再起動を必要とせずに起動時の GUI (デフォルト、ミニ、MPC) が変更できるようになりました。
  • opensubtitles の sub ファイルが再び使えるようになりました。
  • (YouTube) 短い URL (例: http://y2u.be/F5OcZBVPwOA) を認識するようになりました。
  • ビデオファイルのチャプターのサポートが向上しました。
  • バグ修正: お気に入りメニューやコマンドラインからリモート上の m3u ファイルが使えるようになりました。
  • 単一インスタンスの内部処理が QtSingleApplication を使用するように変更されました。

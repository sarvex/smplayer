最新のバージョンは 0.8.0 です。

SMPlayer 0.8.0:

  • ツールバーエディターが追加されました。これにより、ツールバーに表示したいボタンやコントロールが選択できるようになりました。(動画を見る)
  • ビデオフィルター [gradfun]、[ブラー]、[シャープ] が新たに追加されました。
  • 再起動を必要とせずに起動時の GUI (デフォルト、ミニ、MPC) が変更できるようになりました。
  • opensubtitles の sub ファイルが再び使えるようになりました。
  • (YouTube) 短い URL (例: http://y2u.be/F5OcZBVPwOA) を認識するようになりました。
  • ビデオファイルのチャプターのサポートが向上しました。
  • バグ修正: お気に入りメニューやコマンドラインからリモート上の m3u ファイルが使えるようになりました。
  • 単一インスタンスの内部処理が QtSingleApplication を使用するように変更されました。

YouTube ブラウザーの最新のバージョンは 1.1 です。

smtube 1.1:

  • (Linux) 使用するプレーヤーが SMPlayer、MPlayer、VLC、Dragon Player、Totem から選択できるようになりました。
  • Ubuntu において Ambiance または Radiance テーマの使用時にクラッシュする可能性があった問題が修正されました。
  • バスク語訳とポルトガル語訳が追加されました。
  • (Windows) ポータブルバージョンがビルドできるようになりました。

最新のバージョンは 0.7.1 です。このバージョンには重要なバグ修正が含まれていますので、更新を強くおすすめします。

0.7.1:

  • 修正:
    • 初回起動時に SMPlayer の表示まで 10 秒以上かかるのを修正しました。
    • OpenSubtitles からの字幕のダウンロードへのリンクが間違っていました。
    • KDE の [開く] ダイアログが使用されていた場合にお気に入りエディターでファイルを選択しようとすると SMPlayer がクラッシュするのを修正しました。
    • Windows パッケージに一部の翻訳が入っていなかったのを修正しました。
  • 更新:
    • スクリーンショットのデフォルトの保存先が SMPlayer の設定フォルダーからユーザーのピクチャフォルダーになりました。
    • OpenSubtitles のサーバーが変更できるようになりました。
    • YouTube: FLV ではシークが遅くなるので、FLV は優先順位の最低になりました。
    • YouTube: YouTube 動画の検索やダウンロードが可能になりました。これは外部アプリケーションによって提供されます (Linux では smtube という独立したパッケージをインストールすることになります)。

    0.7.0:

    • お気に入りメニューが新たに追加されました。ここで動画、音楽、ストリーム、YouTube 動画などを追加できます。サブメニューも追加できます。
    • youtube に対応しました。[開く] -> [URL] ダイアログを使用するか、ブラウザーから SMPlayer ウィンドウにリンクをドラッグして http://www.youtube.com/watch?v=..... のような URL を追加できます。
    • mplayer2 (http://www.mplayer2.org) に対応しました。これは正確なシーク、MKV 動画でのチャプター順序、優れた一時停止の制御 (停止中のシークなど) といった魅力的な機能を備えた mplayer のフォークです。
    • クロアチア語訳が追加されました。

    0.6.10:

    • vdpau の構成ダイアログが新たに追加されました。使用する vdpau コーデックを選択できます。
    • eCS、OS/2 のポートが追加されました (Silvan Scherrer 氏による)。
    • クローズド キャプションのチャンネルを選択するメニューが新たに追加されました (mplayer r32607 以降を必要とします)。
    • シーク方法 (絶対シークと相対シーク) を選択できるようになりました。
    • プレイリストのアイテムをソートできるようになりました。

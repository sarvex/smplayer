目前最新版本是 0.8.3.

SMPlayer 0.8.3:

  • 修复 YouTube 播放.

SMPlayer 0.8.2:

  • 新增可换肤界面. 还包括几个皮肤.
  • 支持 6.1 和 7.1 音频输出.
  • 在首选项对话框中增加了一个 "隐私" 部分. 您可以禁用保存最近的文件或 URL.
  • 可以通过拖动视频区域来移动主窗口.
  • 修复在 Windows XP 中可能会发生冻结.
  • (Windows) 在显示 GUI 之前创建字体缓存.

SMPlayer 0.8.1:

  • 修复 YouTube 播放.

SMPlayer 0.8.0:

  • 新增工具栏编辑器. 现在可以选择要出现在工具栏上的按钮和控件. (见视频)
  • 新的视频过滤器: 去带(Gradfun)、模糊和锐化.
  • 现在可以在运行时更改 GUI (默认、迷你、MPC), 无需重新启动.
  • OpenSubtitles 的 sub 文件应该重新开始工作了.
  • (YouTube) 识别短网址 (像这样的: http://y2u.be/F5OcZBVPwOA).
  • 更好地支持视频文件的章节.
  • Bug 修复: 远程 m3u 文件可从收藏菜单或命令行下工作.
  • 单实例选项的内部更改 (改用 QtSingleApplication).

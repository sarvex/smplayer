最新版本是 0.8.0。

SMPlayer 0.8.0：

  • 新增工具栏编辑器。现在可以选择要出现在工具栏上的按钮和控件。（见视频）
  • 新的视频过滤器：去带（Gradfun）、模糊和锐化。
  • 现在可以在运行时更改图形用户界面（默认、迷你、MPC），无需重新启动。
  • OpenSubtitles 的 sub 文件应该重新开始工作了。
  • （Youtube）识别短网址（像这样的：http://y2u.be/F5OcZBVPwOA）。
  • 更好地支持视频文件的章节。
  • 错误修复：远程 m3u 文件可从收藏菜单或命令行下工作。
  • 单个实例选项的内部更改（改用 QtSingleApplication）。

YouTube 浏览器的最新版本是 1.1。

SMTube 1.1：

  • （Linux）可以选择要使用的播放器：SMPlayer、MPlayer、VLC、Dragon Player 和 Totem。
  • 修复在 Ubuntu 上如果使用 Ambiance 或 Radiance 主题可能发生崩溃。
  • 新的翻译：巴斯克语和葡萄牙语。
  • （Windows）可以构建一个便携版本。

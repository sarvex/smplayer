A versão mais recente é a 0.7.1:

0.7.1:

Esta versão inclui algumas correções importantes. Reomenda-se a atualização para esta versão.

Correções:

  • O SMPlayer demora mais que 10 segundos a ser exibido quando executado pela primeira vez.
  • As ligações ao sítio web Opensubtitles estavam erradas.
  • O SMPlayer terminava no editor de favoritos, ao tentar selecionar um ficheiro se fosse utilizado a caixa de diálogo do KDE.
  • Algumas correções às traduções no instalador Windows.

Novidades:

  • Por definição, as capturas são agora gravadas na pasta de imagens do utilizador em vez da pasta de configuração do SMPlayer.
  • Adicionada a possibilidade de alterar o servidor de legendas do Opensubtitles.
  • Youtube: a procura em vídeos .flv é lenta. Por isso, a partir desta versão aos vídeos .flv é atribuída a prioridade baixa.
  • Youtube: nesta versão foi adicionada a procura e transferência de vídeos no YouTube. Estas operações são efetuadas através de uma aplicação externa (no linux tem que instalar o pacote smtube).

0.7.0:

  • Adicionado um menu de favoritos, no qual pode adicionar os vídeos, músicas, emissões web e vídeos youtube... Também é possível adicionar submenus.
  • Adicionado o suporte ao YouTube. Agora, é possível abrir URLS do tipo http://www.youtube.com/watch?v=..... através do menu Abrir -> URL ou arrastando a ligação do seu navegador para a janela do SMPlayer.
  • Adicionado o suporte ao mplayer2 (http://www.mplayer2.org). O mplayer2 é uma versão otimizada do mplayer e que adiciona diversas funcionalidades como a procura precisa, a organização de capítulos em vídeos mkv e uma melhor gestão das pausas de vídeo (a procura funciona, mesmo que o vídeo esteja em pausa).
  • Adicionada a tradução em croata.

0.6.10:

  • Adicionada uma caixa de diálogo para configurar os codificadores que o controlador vdpau deve utilizar.
  • Adicionada uma versão para eCS, OS/2 (por Silvan Scherrer).
  • Adicionado um menu para escolher o canal de legendas ocultas. (requer mplayer >= r32607).
  • Adicionada a possibilidade de escolher o método de procura (absoluta ou relativa).
  • Adicionada a possibilidade de organizar os itens da lista de reprodução.

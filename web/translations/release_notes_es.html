La versión más reciente es la 0.7.1.

0.7.1:

This version include some bug fixes, some of them important. It's highly recommended to update.

Fixes:

  • SMPlayer took more than 10 seconds to show when running for the very first time.
  • The links to download subtitles from Opensubtitles were wrong.
  • SMPlayer crashed in the favorite editor when trying to select a file if the KDE open dialog was used.
  • Some translations were missing in the Windows package.

New:

  • By default the screenshots are saved in the user's pictures folder instead of the SMPlayer's config folder.
  • Now it's possible to change the opensubtitles server.
  • Youtube: seeking is slow with flv videos, so now flv videos have the lowest priority.
  • Youtube: now it's possible to search and download videos from youtube. This is provided by an external application (in linux you have to install an independent package: smtube).

0.7.0:

  • Nuevo menú "favoritos", donde puedes añadir tus vídeos favoritos, música, streams, vídeos de youtube... También es posible añadir submenús.
  • Añadido soporte para youtube. Ahora se puede abrir enlaces del tipo http://www.youtube.com/watch?v=..... mediante la opción Abrir -> URL o arrastrando un enlace del navegador a la ventana del smplayer.
  • Añadido soporte para mplayer2 (http://www.mplayer2.org). Se trata de una versión modificada del mplayer que incluye opciones interesantes, como búsqueda precisa, "ordered chapters" de vídeos mkv y mejor manejo de la pausa (por ejemplo es posible desplazarse por el vídeo mientras se encuentra en pausa).
  • Se ha añadido la traducción al croata.

0.6.10:

  • Nuevo diálogo de configuración para vdpau, que permite seleccionar los códecs vdpau a usar.
  • Versión para eCS, OS/2 (por Silvan Scherrer).
  • Nuevo menú para seleccionar el canal de los "closed captions" (requiere mplayer >= r32607).
  • Posibilidad de seleccionar el método de búsqueda (absoluto or relativo).
  • Posibilidad de ordenar los ficheros de la lista de reproducción.

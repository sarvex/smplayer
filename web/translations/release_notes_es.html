smplayer 0.8.4:

  • Nueva opción que permite cambiar los fps (cuadros por segundo) de los subtítulos externos.
  • El ecualizador de vídeo se ha reescrito.
  • Ahora smplayer comprueba automáticamente si hay actualizaciones e informa al usuario si se encuentra una versión nueva.
  • Añadido soporte para ISO-8859-16 para los subtítulos.
  • The support for MPlayer2 has been improved and now we provide an optional package for Windows which includes MPlayer2. MPlayer2 has interesting new features like support for mkv ordered chapters and precise seeks.
  • Nuevas traducciones: tailandés y hebreo.
  • Se han corregido algunos bugs.

smplayer 0.8.3:

  • La reproducción de vídeos de Youtube vuelve a funcionar.

smplayer 0.8.2:

  • Se ha añadido una interfaz con soporte de pieles (temas). Se incluyen varias pieles.
  • Soporte para salida de audio 6.1 y 7.1.
  • Se ha añadido una sección "privacidad" en el diálogo de preferencias, que permite evitar que se guarde información sobre ficheros recientes y URLs.
  • La ventana principal se puede mover arrastrando la zona de vídeo.
  • Se evita una ralentización que podía ocurrir en Windows XP.
  • (Windows) La caché de tipos de letras se crea antes de iniciar el GUI.

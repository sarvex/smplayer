The latest version is 0.7.1.

0.7.1:

Эта версия включает несколько исправлений, в том числе важных. Настоятельно рекомендуется обновиться.

Исправления:

  • SMPlayer показывался только через 10 секунд при первом запуске.
  • Ссылки на загрузку субтитров с Opensubtitles были неверными.
  • SMPlayer падал в редакторе списка избранного при попытке выделить файл при использовании диалога открытия файлов KDE.
  • Некоторые переводы отсутствовали в пакете для Windows.

Новое:

  • По умолчанию скриншоты сохраняются в пользовательский каталог изображений вместо указанного каталога с настройками SMPlayer.
  • Теперь стало возможным изменение сервера opensubtitles.
  • Youtube: перемотка медленная в случае flv файлов, поэтому flv видеоролики теперь имеют минимальный приоритет.
  • Youtube: теперь стало возможным искать и загружать видео с youtube. Это реализовано в виде внешнего приложения (в linux вы должны установить независимый пакет: smtube)

0.7.0:

  • Новое меню избранного, где можно добавить любимые видеофайлы, музыкальные файлы, потоки, видео youtube... Также можно добавлять подменю.
  • Поддержка youtube. Теперь можно открывать url вида http://www.youtube.com/watch?v=....., используя диалог Открыть -> Адрес или перетащив ссылку из браузера на окно smplayer.
  • Поддержка mplayer2 (http://www.mplayer2.org). Это форк mplayer с интересными возможностями вроде точной перемотки, линкованными mkv видеофайлами и лучшей обработкой пауз (например, перемотка работает и в случае паузы).
  • Новый перевод: хорватский.

0.6.10:

  • Новый диалог настройки vdpau, позволяющий выбрать используемые кодеки vdpau.
  • Порт на eCS, OS/2 (от Silvan Scherrer).
  • Новое меню выбора канала закрытых субтитров (требуется mplayer >= r32607).
  • Возможность выбрать метод перемотки (абсолютный или относительный).
  • Возможность сортировки элементов в плейлисте.
